提示

因毒盘阻拦获取资源，请先修改UA:

IDM（电脑）

  1. 选项 -> 下载 -> 手动添加任务时使用的用户代理（UA）-> 填入 LogStatistic
  2. 右键复制下载链接（直接点击 或 右键调用 IDM，将会导致下载失败），在 IDM 新建任务，粘贴链接即可下载。

  3. ADM(电脑)点击下载

ADM或IDM+（Android）

  1. ADM:设置 –> 下载中 -> 浏览器标识(自定义)
  2. IDM+:设置–>高级设置–>用户代理–>填入自定义uA
  3. 自定义UA：LogStatistic

  4. ADM点击下载    &&   IDM+美化版（点击了解详情）

Alook 浏览器（IOS）

  1. 设置 -> 通用设置 -> 浏览器标识 -> 移动版浏览器标识 -> 自定义 ->
    填入 LogStatistic
