# 安装【Tampermonkey】

Q：什么是Tampermonkey？

A：Tampermonkey，俗称“油猴”，是一款在览器上运行和管理脚本的插件。所谓脚本就是一段代码，它们能够优化您的网页浏览体验。安装之后，有些脚本能为网站添加新的功能，有些能使网站的界面更加易用，有些则能隐藏网站上烦人的部分内容。
